1. Which among the following are active connected entities in a holistic smart city environment?

  • a. Police station
  • b. Banks
  • c. Transport centers
  • d. All of the above

Correct Answer: d. All of the above

2. With the help of ICT tools, it is possible to increase and improve citizen participation for a good governance based smart city

  • a. True
  • b. False

Correct Answer: a. True

3. Which among the following is a possible challenge with respect to smart parking lots?

  • a. Efficient auto-routing of vehicles
  • b. Locating current vacant spots
  • c. Auto-charging of vehicles

Correct Answer: d. All of the given

4. What refers to combining information from multiple sensor sources?

  • a. Information Collection

Correct Answer: d. Multi-sensor data fusion

5. Which of the following is one of the theory of evidence-based mathematical methods of data fusion?

  • a. Belief function

Correct Answer: a. Belief function

6. With respect to data fusion from multiple IoT sensors, does outlier data present with a challenge?

7. Which of the following is a phase of ICV development?

  • a. Based on 2G
  • b. Based on 4G LTE
  • c. Vehicles connected to cloud
  • d. All of these

Correct Answer: d. All of these

  • a. SDN

9. Which of the following statements are true about the HAN standards?

  • Statement I: Physical and MAC layers are defined by IEEE802.15.4.
  • Statement II: Network layer is defined by Zigbee.

  • b. Statements I and III

10. What is UPnP?
